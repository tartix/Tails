Tails 3.7.1 is out 2018-06-10

This release fixes many security issues and users should upgrade as soon as possible.

  1. Cambi
  2. Problemi noti
  3. Get Tails 3.7.1
  4. Cosa arriverà nelle prossime versioni?

Cambi

Aggiornamenti e cambiamenti

  • Upgrade Tor Browser to 7.5.5.

    This fixes a critical issue in Firefox announced on June 6.

  • Upgrade Thunderbird to 52.8.0.

    This partly mitigates the infamous EFAIL attack and fixes importing OpenPGP keys from keyservers with Enigmail.

Problemi risolti

  • Fix setting a screen locker password with non-ASCII characters. (#15636)

  • Rename the WhisperBack launcher to "WhisperBack Error Reporting" so that users have a better chance to understand what it does. (#6432)

Per maggiori dettagli, leggi il nostro changelog.

Problemi noti

Nessuno che riguardi questo specifico rilascio.

Vedi la lista dei problemi noti da tempo.

Get Tails 3.7.1

  • Per installare, segui le nostre istruzioni per l'installazione.

  • To upgrade, automatic upgrades are available from 3.6, 3.6.1, 3.6.2, and 3.7 to 3.7.1.

    Se non potete fare un aggiornamento automatico o se Tails non viene avviato dopo un aggiornamento automatico, provate a fare un aggiornamento manuale.

  • Download Tails 3.7.1.

Cosa arriverà nelle prossime versioni?
