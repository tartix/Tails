Tails 4.4 is out 2020-03-12

This release fixes many security vulnerabilities. You should upgrade as soon as possible.

  1. Changes and upgrades
  2. Problemi risolti
  3. Problemi noti
  4. Get Tails 4.4
  5. Cosa arriverà nelle prossime versioni?

Changes and upgrades

  • Update Tor Browser to 9.0.6.

  • Update Thunderbird to 68.5.0.

  • Update Linux to 5.4.19. This should improve the support for newer hardware (graphics, Wi-Fi, etc.).

Problemi risolti

  • Tentatively fix Wi-Fi interfaces with Realtek RTL8822BE and RTL8822CE chipsets. (#17323)

    If you had a problem with Wi-Fi starting from Tails 4.1, please let us know whether your problem was fixed or not.

Per maggiori dettagli, leggi il nostro changelog.

Problemi noti

JavaScript sometimes enabled in the Safest security level of Tor Browser

Tor è al corrente di un problema tecnico che permette l'esecuzione di JavaScript nel livello di sicurezza Molto sicuro di Tor Browser in certe situazioni.

Fino ad allora, fai ciò che segue per disattivare completamente JavaScript invece di impostare il livello di sicurezza:

  1. Consulta l'indirizzo about:config in Tor Browser.

  2. Fai clic sul bottone Accetto i rischi.

  3. Nella casella in cima alla pagina, cerca javascript.enabled.

  4. Double-click on the javascript.enabled line in the results to set its value to false.

Vedi la lista dei problemi noti da tempo.

Get Tails 4.4

Per aggiornare la propria chiavetta USB Tails e conservare il proprio volume persistente

  • Automatic upgrades are available from 4.2, 4.2.2, and 4.3 to 4.4.

  • Se non puoi eseguire un aggiornamento automatico o se Tails non si avvia dopo un aggiornamento automatico, ti preghiamo di provare a eseguire un aggiornamento manuale.

Per installare Tails su una nuova chiavetta USB

Follow our installation instructions:

  • Installare da Windows
  • Installare da macOS
  • Installare da Linux

All the data on this USB stick will be lost.
