Ha salido Tails 3.6.1 2018-03-18

This release fixes several security issues and users should upgrade as soon as possible.

  1. Cambios
  2. Problemas conocidos
  3. Obtén Tails 3.6.1
  4. ¿Qué novedades hay?

Cambios

Actualizaciones y cambios

  • Actualización a Tor Browser 7.5.2.

    This fixes a critical issue in Firefox announced on March 16.

Problemas arreglados

  • This release is reproducible, unlike Tails 3.6. (#15400)

Para más detalles, lee nuestro changelog.

Problemas conocidos

  • Tails Verification, our browser extension to verify Tails downloads, is broken in Tails 3.6.1. Please use it from Firefox, Chrome, or Tor Browser outside of Tails. (#15434)

Mira la lista de problemas duraderos.

Obtén Tails 3.6.1

  • Para instalarlo, sigue nuestras instrucciones de instalación.

  • Para actualizar, están disponibles actualizaciones automáticas de 3.6 a 3.6.1.

    Si no puedes hacer una actualización automática o si Tails falla al iniciar despues de una actualización automática, por favor intenta hacer una actualización manual.

    Si estás haciendo una actualización manual desde Tails 3.2, 3.3, 3.4, o 3.5, es posible seleccionar una imagen ISO cuando ejecutas Tails en inglés. Para otros idiomas, el botón del explorador de archivos queda deshabilitado:

    Para hacer una actualización manual desde Tails 3.2, 3.3, 3.4, o 3.5, puedes:

    • Reiniciar Tails en ingles.

    • Iniciar Tails Installer en ingles desde la linea de comandos:

      LANG=en_US tails-installer

  • Download Tails 3.6.1.

¿Qué novedades hay?

Tails 3.7 está planificado para el 8 de Mayo.

Echa un vistazo a nuestro roadmap para ver hacia dónde nos dirigimos.

We need your help and there are many ways to contribute to Tails (donating is only one of them). Come talk to us!
