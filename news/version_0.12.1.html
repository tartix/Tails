Tails 0.12.1 is out 2012-07-01

Tails, The Amnesic Incognito Live System, version 0.12.1, is out.

This is a brown paper bag release aimed at fixing two major problems introduced in Tails 0.12. While upgrading is recommended, it's not strictly necessary for users that haven't experienced any issues with Tails 0.12.

  1. Changes
  2. I want to try it / to upgrade!
  3. A glimpse towards the future

Changes

  • Iceweasel

    • Upgrade Torbutton to 1.4.6.
    • Upgrade AdBlock Plus to 2.1.
    • Update AdBlock patterns.

  • Hardware support

    • Upgrade Linux to 3.2.21-3 (linux-image-3.2.0-3-486).

  • Software

    • Install python-pdfrw to re-add PDF support to the MAT.
    • Upgrade tails-greeter to 0.7.1, which fixes the race condition that broke administration password and locale settings on some systems.

  • Boot

    • Remove the Tails specific plymouth theme. The theme interfers heavily with the boot process on some hardware.

Plus the usual bunch of minor bug reports and improvements.

See the online Changelog for technical details.
