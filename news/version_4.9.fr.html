Tails 4.9 is out 2020-07-28

This release fixes many security vulnerabilities. You should upgrade as soon as possible.

  1. Changements et mises à jour
  2. Problèmes corrigés
  3. Problèmes connus
  4. Get Tails 4.9
  5. Et ensuite ?

Changements et mises à jour

  • Update Tor Browser to 9.5.3.

  • Update Thunderbird to 68.10.0.

  • Update Linux to 5.7.6. This should improve the support for newer hardware (graphics, Wi-Fi, etc.).

Problèmes corrigés

  • Allow characters others than A–Z, a–z, 1–9, and _@%+=:,./- in the administration password. (#17792)

  • Apply the keyboard layout that is automatically selected when you change the language in the Welcome Screen. (#17794)

  • Fix starting Tails with the toram boot option. (#17800)

Pour plus de détails, lisez notre liste des changements.

Problèmes connus

  • USB Wi-Fi adapters with Atheros AR9271 hardware do not work with Linux 5.7.6. (#17834)

Voir la liste des problèmes connus de longue date.

Get Tails 4.9

Pour mettre à jour votre clé USB Tails et conserver votre stockage persistant

  • Automatic upgrades are available from Tails 4.2 or later to 4.9.

  • Si vous ne pouvez pas faire une mise à jour automatique ou si le démarrage de Tails échoue après une mise à jour automatique, merci d'essayer de faire une mise à jour manuelle.

Pour installer Tails sur une nouvelle clé USB

Suivez nos instructions d'installation :

  • Installer depuis Windows
  • Installer depuis macOS
  • Installer depuis Linux

Le stockage persistant de la clé USB sera perdu si vous faites une installation au lieu d'une mise à jour.

Pour seulement télécharger

If you don't need installation or upgrade instructions, you can download Tails 4.9 directly:

  • Pour clés USB (image USB)
  • Pour DVD et machines virtuelles (image ISO)

Et ensuite ?

Tails 4.10 is scheduled for August 25.

Jetez un œil à notre feuille de route pour savoir ce que nous avons en tête.

We need your help and there are many ways to contribute to Tails (donating is only one of them). Come talk to us!
