Ha salido Tails 4.2 2020-01-07

Esta versión corrige muchos problemas de seguridad y los usuarios deberían actualizar lo más pronto posible.

Mejoras en las actualizaciones automáticas

Trabajamos en mejoras de la funcionalidad de actualización automática, que es todavía uno de los momentos más dolorosos cuando usas Tails:

  • Hasta ahora, si tu versión de Tails tenía varios meses, tenías que hacer 2 o más actualizaciones automáticas seguidas. Por ejemplo, para actualizar desde Tails 3.12 a Tails 3.16, primero tenías que actualizar a Tails 3.14.

    A partir de Tails 4.2, habrá updates directos para todas las versiones anteriores a la actual.

  • Hasta ahora sólo podías hacer un número limitado de actualizaciones automáticas, a partir de cierto número tenías que hacer la actualización "manual", mucho más complicada.

    A partir de Tails 4.2, sólo tendrás que hacer actualizaciones manuales entre versiones mayores, por ejemplo para actualizar a Tails 5.0 en 2021.

  • Hicimos que las actualizaciones automáticas usen menos memoria.

  • Optimizamos un poco el tamaño de la descarga de las actualizaciones automáticas.

Nuevas funcionalidades

  • We included several command line tools used by SecureDrop users to analyze the metadata of leaked documents on computers that cannot use the Additional Software feature:

    • PDF Redact Tools to redact and strip metadata from text documents before publishing
    • Tesseract OCR to convert images containing text into a text document
    • FFmpeg to record and convert audio and video

Cambios y actualizaciones

  • Update Tor Browser to 9.0.3.

  • Update Thunderbird to 68.3.0.

  • Update Linux to 5.3.15.

Problemas arreglados

  • Open ~/Persistent/keepassx.kdbx by default when starting KeePassX. If this database does not exist yet, stop pointing to it in the list of recent databases.

Para más detalles, lee nuestro changelog.

Problemas conocidos

Nada concreto para esta versión.

Mira la lista de problemas duraderos.

Get Tails 4.2

Para actualizar Tails y mantener tu almacenamiento persistente

  • Automatic upgrades are available from 4.0, 4.1, and 4.1.1 to 4.2.

  • Si no puedes hacer una actualización automática, o si Tails falla al iniciar después de una actualización automática, intenta hacer una actualización manual.
