Tails 4.9 is out 2020-07-28

This release fixes many security vulnerabilities. You should upgrade as soon as possible.

  1. Modifiche e aggiornamenti
  2. Problemi risolti
  3. Problemi noti
  4. Get Tails 4.9
  5. Cosa arriverà nelle prossime versioni?

Modifiche e aggiornamenti

  • Update Tor Browser to 9.5.3.

  • Update Thunderbird to 68.10.0.

  • Update Linux to 5.7.6. This should improve the support for newer hardware (graphics, Wi-Fi, etc.).

Problemi risolti

  • Allow characters others than A–Z, a–z, 1–9, and _@%+=:,./- in the administration password. (#17792)

  • Apply the keyboard layout that is automatically selected when you change the language in the Welcome Screen. (#17794)

  • Fix starting Tails with the toram boot option. (#17800)

Per maggiori dettagli, leggi il nostro changelog.

Problemi noti

  • USB Wi-Fi adapters with Atheros AR9271 hardware do not work with Linux 5.7.6. (#17834)

Vedi la lista dei problemi noti da tempo.

Get Tails 4.9

Per aggiornare la propria chiavetta USB Tails e conservare il proprio volume persistente

  • Automatic upgrades are available from Tails 4.2 or later to 4.9.

  • Se non puoi eseguire un aggiornamento automatico o se Tails non si avvia dopo un aggiornamento automatico, ti preghiamo di provare a eseguire un aggiornamento manuale.

Per installare Tails su una nuova chiavetta USB

Follow our installation instructions:

  • Installare da Windows
  • Installare da macOS
  • Installare da Linux

The Persistent Storage on the USB stick will be lost if you install instead of upgrading.

To download only

If you don't need installation or upgrade instructions, you can download Tails 4.9 directly:

  • Per chiavette USB (immagine USB)
  • For DVDs and virtual machines (ISO image)

Cosa arriverà nelle prossime versioni?
