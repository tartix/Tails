Chatting with Dino and OMEMO
← Enabling a wireless device | advanced topics.index

Dino is a modern, open-source Jabber/XMPP chat client. Dino supports end-to-end encryption using OMEMO.

To learn how to install and use Dino in Tails, see this tutorial.

Dino is being considered as an option to replace Pidgin in Tails.

Please report your findings about using Dino in Tails on:

  • tails-testers@boum.org (public mailing list)
  • #8573
