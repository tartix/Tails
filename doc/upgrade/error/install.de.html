Fehler bei der Installation der Aktualisierung

Die Aktualisierung konnte nicht installiert werden.

Your Tails USB stick needs to be repaired and might be unable to restart.

Dies wird möglicherweise durch einen Softwarefehler im Tails Upgrader verursacht.

Reporting the problem to our help desk

If you are comfortable with the command line, please report the problem to our help desk.

Include in your report:

  • die Zusatzinformationen, die mit der Fehlermeldung angezeigt werden
  • die Ausgabe der folgenden fünf Befehle, ausgeführt in einem Terminal: 
        ls -l /lib/live/mount/medium/live
    cat /lib/live/mount/medium/live/Tails.module
    mount
    df -h
    free -m

Repairing your Tails USB stick

To repair your Tails USB stick, please try doing a manual upgrade.

The Persistent Storage on the USB stick will be preserved.

Wenn die manuelle Aktualisierung ebenfalls fehlschlägt, senden Sie uns bitte einen weiteren Fehlerbericht.
