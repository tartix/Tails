Ошибка при установке обновления

Обновление не устанавливается.

Ваша флешка Tails сломалась. Возможно, с неё нельзя перезагрузить систему.

Возможно, это связано с программной ошибкой в Tails Upgrader.

Как сообщить о проблеме в техподдержку

Если вы умеете обращаться с командной строкой, пожалуйста, сообщите о проблеме нашей службе техподдержки.

Включите в своё сообщение:

  • отладочную информацию из сообщения об ошибке;
  • результаты выполнения следующих пяти команд, выполненных в терминале: 
        ls -l /lib/live/mount/medium/live
    cat /lib/live/mount/medium/live/Tails.module
    mount
    df -h
    free -m

Восстановление флешки Tails

Для восстановления флешки Tails нужно выполнить обновление вручную.

Постоянное хранилище на флешке сохранится.

Если и вручную обновиться не выходит, пожалуйста, напишите нам об этой ошибке.
