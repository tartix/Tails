Numerous security holes in Tails 3.12.1 2019-03-18

Several security holes that affect Tails 3.12.1 are now fixed in Tails 3.13.

We strongly encourage you to upgrade to Tails 3.13 as soon as possible.

  • Tor Browser: Mozilla Foundation Security Advisory 2019-07
  • Thunderbird: Mozilla Foundation Security Advisory 2019-06
  • Linux:
    • CVE-2019-8956
    • CVE-2019-8912
    • CVE-2019-9162
    • CVE-2019-9213
    • CVE-2019-8980
    • CVE-2019-3701
    • CVE-2019-7308
    • CVE-2018-16880
    • CVE-2019-6974
    • CVE-2019-7222
    • CVE-2019-7221
    • CVE-2019-3819
  • LDB: Debian Security Advisory 4397
  • OpenJPEG: Debian Security Advisory 4405
  • OpenSSL 1.0: Debian Security Advisory 4400
  • OpenSSH: Debian Security Advisory 4387
