Numerous security holes in Tails 4.11 2020-10-17

Tails 4.12 fixes many security issues that affect Tails 4.11. You should upgrade to Tails 4.12 as soon as possible.

  • Linux: CVE-2020-25285, CVE-2020-25284, CVE-2020-14390, CVE-2020-14314, CVE-2020-14386
  • Tor Browser Mozilla Foundation Security Advisory 2020-46
