Numerous security holes in Tails 0.13 2012-11-08

Several security holes affect Tails 0.13.

We strongly urge you to upgrade to Tails 0.14 as soon as possible in case you are still using an older version.

Details

  • iceweasel (Debian Security Advisory 2565, Debian Security Advisory 2553)
  • openoffice.org (Debian Security Advisory 2570)
  • tiff (Debian Security Advisory 2561, Debian Security Advisory 2552)
  • libexif (Debian Security Advisory 2559)
  • libxslt (Debian Security Advisory 2555)
  • libproxy (Debian Security Advisory 2571)
