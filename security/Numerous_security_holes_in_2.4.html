Numerous security holes in Tails 2.4 2016-07-30

Several security holes that affect Tails 2.4 are now fixed in Tails 2.5.

We strongly encourage you to upgrade to Tails 2.5 as soon as possible.

Details

  • Tor Browser: Mozilla Foundation Security Advisory 2016-62, Mozilla Foundation Security Advisory 2016-63, Mozilla Foundation Security Advisory 2016-64, Mozilla Foundation Security Advisory 2016-65, Mozilla Foundation Security Advisory 2016-67, Mozilla Foundation Security Advisory 2016-70, Mozilla Foundation Security Advisory 2016-72, Mozilla Foundation Security Advisory 2016-73, Mozilla Foundation Security Advisory 2016-76, Mozilla Foundation Security Advisory 2016-77, Mozilla Foundation Security Advisory 2016-78, Mozilla Foundation Security Advisory 2016-79, Mozilla Foundation Security Advisory 2016-80
  • Icedove: Debian Security Advisory 3601
  • Linux: Debian Security Advisory 3616, Debian Security Advisory 3607
  • GIMP: Debian Security Advisory 3612
  • libav: Debian Security Advisory 3603
  • expat: Debian Security Advisory 3597
  • libgd3: Debian Security Advisory 3630
  • libmodule-build-perl, perl: Debian Security Advisory 3628
  • Pidgin: Debian Security Advisory 3620 (Tails is not affected since it does not include support for the MXit protocol)
  • LibreOffice: Debian Security Advisory 3608
  • libxslt1.1: Debian Security Advisory 3605
  • OpenSSH: Debian Security Advisory 3626 (Tails is not affected since it does not include the SSH daemon)
  • p7zip: Debian Security Advisory 3599
