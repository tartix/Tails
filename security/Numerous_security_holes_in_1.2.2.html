Numerous security holes in Tails 1.2.2 2015-01-12

Several security holes that affect Tails 1.2.2 are now fixed in Tails 1.2.3.

We strongly encourage you to upgrade to Tails 1.2.3 as soon as possible.

Details

  • Tails' MAC spoofing feature may leak the real MAC address (#8571).
  • The Unsafe Browser checks for an upgrade of the Tor Browser in the clear (#8694).
  • Tor Browser and its bundled NSS: Mozilla Foundation Security Advisory 2015-01, Mozilla Foundation Security Advisory 2015-03, Mozilla Foundation Security Advisory 2015-04, Mozilla Foundation Security Advisory 2015-06
  • tcpdump: Debian Security Advisory 3086
  • linux: Debian Security Advisory 3093
  • bind9: Debian Security Advisory 3094
  • xorg-server: Debian Security Advisory 3095
  • dbus: Debian Security Advisory 3099
  • libyaml: Debian Security Advisory 3102
  • libyaml-libyaml-perl: Debian Security Advisory 3103
  • cpio: Debian Security Advisory 3111
  • unzip: Debian Security Advisory 3113
  • mime-support: Debian Security Advisory 3114
  • libevent: Debian Security Advisory 3119
  • file: Debian Security Advisory 3121
  • curl: Debian Security Advisory 3122
  • binutils: Debian Security Advisory 3123
  • openssl: Debian Security Advisory 3125
