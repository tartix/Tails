Numerous security holes in Tails 2.12 2017-06-08

Several security holes that affect Tails 2.12 are now fixed in Tails 3.0.

We strongly encourage you to upgrade to Tails 3.0 as soon as possible.

  • Tor Browser: Mozilla Foundation Security Advisory 2017-16
  • MAC spoofing has been fixed on some network interfaces (TP-Link WN725N): (#12362)
  • Linux: 2016-2188, 2016-9604, 2017-0605, 2017-1000363, 2017-2636, 2017-2671, 2017-6874, 2017-6874, 2017-6951, 2017-7184, 2017-7184, 2017-7187, 2017-7261, 2017-7294, 2017-7308, 2017-7374, 2017-7472, 2017-7477, 2017-7487, 2017-7616, 2017-7618, 2017-7645, 2017-7889, 2017-7895, 2017-8061, 2017-8063, 2017-8064, 2017-8067, 2017-8890, 2017-9059, 2017-9074, 2017-9075, 2017-9076, 2017-9077, 2017-9150, 2017-9211, 2017-9242
