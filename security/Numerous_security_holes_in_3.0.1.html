Numerous security holes in Tails 3.0.1 2017-08-05

Several security holes that affect Tails 3.0.1 are now fixed in Tails 3.1.

We strongly encourage you to upgrade to Tails 3.1 as soon as possible.

  • Linux: Debian Security Advisory 3927
  • MSFA 2017-19
  • tiff: Debian Security Advisory 3903
  • bind9: Debian Security Advisory 3904
  • evince: Debian Security Advisory 3911
  • imagemagick: Debian Security Advisory 3914
