Numerous security holes in Tails 0.9 2012-01-01

The following security holes affect Tails 0.9.

We strongly urge you to upgrade to Tails 0.10 as soon as possible in case you are still using an older version.

Details

These are Debian security announces; details can be found on the Debian security page:

  • jasper (DSA-2371)
  • libsoup2.4 (DSA-2369)
  • xorg (DSA-2364)
  • tor (DSA-2363)
  • openjdk-6 (DSA-2358)
  • evince (DSA-2357)
  • cups (DSA-2354)
  • freetype (DSA-2350)
