Numerous security holes in Tails 4.10 2020-09-19

Tails 4.11 fixes many security issues that affect Tails 4.10. You should upgrade to Tails 4.11 as soon as possible.

  • Tor Browser: Mozilla Foundation Security Advisory 2020-43
  • Thunderbird: Mozilla Foundation Security Advisory 2020-40
  • Linux: CVE-2020-16166, CVE-2020-14331, CVE-2019-19770
  • ghostscript: Debian Security Advisory 4748
  • bind9: Debian Security Advisory 4752
  • openexr: Debian Security Advisory 4755
  • xorg-server: Debian Security Advisory 4758
  • libzmq5: Debian Security Advisory 4761
