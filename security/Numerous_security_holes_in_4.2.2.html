Numerous security holes in Tails 4.2.2 2020-02-09

Tails 4.3 fixes many security issues that affect Tails 4.2.2. You should upgrade to Tails 4.3 as soon as possible.

  • Tor Browser: Mozilla Foundation Security Advisory 2020-06
  • Linux: CVE-2019-19965, CVE-2019-18809, CVE-2019-14901, CVE-2019-14895, CVE-2019-19056, CVE-2019-19053, CVE-2019-19066, CVE-2019-19068, CVE-2019-19078, CVE-2019-14615, CVE-2019-19063, CVE-2019-19057, CVE-2019-18786, CVE-2019-19947, CVE-2019-19037, CVE-2019-19071, CVE-2019-19062, CVE-2019-19332, CVE-2019-15099, CVE-2019-18683, CVE-2019-15291, CVE-2019-18660
  • thunderbird: Mozilla Foundation Security Advisory 2020-04, Debian Security Advisory 4603
  • tiff: Debian Security Advisory 4608
  • python-apt: Debian Security Advisory 4609
  • qtbase-opensource-src: Debian Security Advisory 4617
  • libexif: Debian Security Advisory 4618
  • sudo: Debian Security Advisory 4614
  • libidn2: Debian Security Advisory 4613
  • webkit2gtk: Debian Security Advisory 4610
