Numerous security holes in Tails 0.22 2014-02-01

Several security holes affect Tails 0.22.

We strongly encourage you to upgrade to Tails 0.22.1 as soon as possible.

Details

  • Iceweasel and its bundled NSS: Mozilla Foundation Security Advisory 2014-12
  • Linux kernel: 2012-2372, 2013-2147, 2013-2889, 2013-2891, 2013-2893, 2013-2894, 2013-2895, 2013-2897, 2013-2929, 2013-2930, 2013-4299, 2013-4343, 2013-4345, 2013-4348, 2013-4350, 2013-4387, 2013-4563, 2013-4587, 2013-6367, 2013-6368, 2013-6376, 2013-6378, 2013-6382, 2013-6405, 2013-6432, 2014-1438
  • djvulibre: Debian Security Advisory 2844
  • GnuPG: Debian Security Advisory 2821
  • hplip: Debian Security Advisory 2829
  • libtar: Debian Security Advisory 2817
  • Pidgin: 2013-6477, 2013-6478, 2013-6479, 2013-6483, 2013-6484, 2013-6486, 2014-0020, 2014-6485
