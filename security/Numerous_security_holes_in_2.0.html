Numerous security holes in Tails 2.0 2016-02-12

Several security holes that affect Tails 2.0 are now fixed in Tails 2.0.1.

We strongly encourage you to upgrade to Tails 2.0.1 as soon as possible.

Details

  • Tor Browser: Mozilla Foundation Security Advisory 2016-13, Mozilla Foundation Security Advisory 2016-14
  • virtualbox: Debian Security Advisory 3454
  • curl: Debian Security Advisory 3455
  • openjdk-7: Debian Security Advisory 3458
  • krb5: Debian Security Advisory 3466
  • tiff: Debian Security Advisory 3467
