Numerous security holes in Tails 4.4 2020-03-21

Tails 4.4.1 fixes many security issues that affect Tails 4.4. You should upgrade to Tails 4.4.1 as soon as possible.

  • Tor Browser: 9.0.7 release announcement
  • Tor: TROVE-2020-002 (bug #33120 on Tor Project's Trac, CVE-2020-10592), TROVE-2020-003 (bug #33137 on Tor Project's Trac), TROVE-2020-004 (bug #33619 on Tor Project's Trac, CVE-2020-10593)
  • Thunderbird: Mozilla Foundation Security Advisory 2020-10, Debian Security Advisory 4642
  • WebKitGTK: Debian Security Advisory 4641
