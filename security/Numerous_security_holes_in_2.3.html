Numerous security holes in Tails 2.3 2016-06-05

Several security holes that affect Tails 2.3 are now fixed in Tails 2.4.

We strongly encourage you to upgrade to Tails 2.4 as soon as possible.

Details

  • Tor Browser: Mozilla Foundation Security Advisory 2016-49, Mozilla Foundation Security Advisory 2016-50, Mozilla Foundation Security Advisory 2016-51, Mozilla Foundation Security Advisory 2016-52, Mozilla Foundation Security Advisory 2016-53, Mozilla Foundation Security Advisory 2016-55, Mozilla Foundation Security Advisory 2016-56, Mozilla Foundation Security Advisory 2016-58
  • libarchive: Debian Security Advisory 3574
  • icedove: Debian Security Advisory 3576
  • imagemagick: Debian Security Advisory 3580, Debian Security Advisory 3591
  • expat: Debian Security Advisory 3582
  • libgd3: Debian Security Advisory 3556, Debian Security Advisory 3587
  • gdk-pixbuf: Debian Security Advisory 3589
  • libidn: Debian Security Advisory 3578
  • libndp: Debian Security Advisory 3582
  • poppler: Debian Security Advisory 3563
  • librsvg: Debian Security Advisory 3584
  • openssl: Debian Security Advisory 3566
  • libtasn1-6: Debian Security Advisory 3568
  • libxml2: Debian Security Advisory 3593
  • openjdk-7-jre: Debian Security Advisory 3558
