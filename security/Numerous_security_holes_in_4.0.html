Numerous security holes in Tails 4.0 2019-12-01

Tails 4.1 fixes many security issues that affect Tails 4.0. You should upgrade to Tails 4.1 as soon as possible.

  • Tor Browser: Mozilla Foundation Security Advisory 2019-37
  • Thunderbird: Mozilla Foundation Security Advisory 2019-38
  • Linux: CVE-2019-11135, CVE-2018-12207, CVE-2019-0155, CVE-2019-0154, CVE-2019-17133, CVE-2019-17055, CVE-2019-17054, CVE-2019-17052, CVE-2019-17053, CVE-2019-17056, CVE-2019-16746, CVE-2019-17075, CVE-2019-15098
  • file: Debian Security Advisory 4550
  • FriBidi: Debian Security Advisory 4561
  • Ghostscript: Debian Security Advisory 4569
  • Intel microcode: Debian Security Advisory 4565, CVE-2019-0117
  • libarchive: Debian Security Advisory 4557
  • libvpx: Debian Security Advisory 4578
  • libxslt: CVE-2019-18197
  • ncurses: CVE-2019-17594, CVE-2019-17595
  • Python 2.7: CVE-2018-20852, CVE-2019-10160, CVE-2019-16056, CVE-2019-16935, CVE-2019-9740, CVE-2019-9947
  • Qt: Debian Security Advisory 4556
  • tcpdump: Debian Security Advisory 4547
  • WebKitGTK: Debian Security Advisory 4558, Debian Security Advisory 4563
