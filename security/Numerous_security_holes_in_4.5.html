Numerous security holes in Tails 4.5 2020-05-03

Tails 4.6 fixes many security issues that affect Tails 4.5. You should upgrade to Tails 4.6 as soon as possible.

  • Tor Browser: Mozilla Foundation Security Advisory 2020-17
  • Thunderbird: Mozilla Foundation Security Advisory 2020-14
  • Git: Debian Security Advisory 4657, Debian Security Advisory 4659
  • Node.js: Debian Security Advisory 4669
  • OpenLDAP: Debian Security Advisory 4666
  • OpenSSL: Debian Security Advisory 4661
  • ReportLab: Debian Security Advisory 4663
  • WebKitGTK: Debian Security Advisory 4658
