Numerous security holes in Tails 3.9.1 2018-10-22

Several security holes that affect Tails 3.9.1 are now fixed in Tails 3.10.1.

We strongly encourage you to upgrade to Tails 3.10.1 as soon as possible.

  • Tor Browser: Mozilla Foundation Security Advisory 2018-27
  • Thunderbird: Mozilla Foundation Security Advisory 2018-25
  • Linux:

    • CVE-2018-18021

  • libssh:
    • Debian Security Advisory 4322
