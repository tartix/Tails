Numerous security holes in Tails 3.12 2019-02-11

Several security holes that affect Tails 3.12 are now fixed in Tails 3.12.1.

We strongly encourage you to upgrade to Tails 3.12.1 as soon as possible.

  • Tor Browser: Mozilla Foundation Security Advisory 2019-05
  • LibreOffice: Debian Security Advisory 4381
  • cURL: Debian Security Advisory 4386
  • Qt 5: Debian Security Advisory 4374
  • OpenSSH: Debian Security Advisory 4387
