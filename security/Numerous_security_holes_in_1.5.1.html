Numerous security holes in Tails 1.5.1 2015-09-20

Several security holes that affect Tails 1.5.1 are now fixed in Tails 1.6.

We strongly encourage you to upgrade to Tails 1.6 as soon as possible.

Details

  • Tor Browser: Mozilla Foundation Security Advisory 2015-96, Mozilla Foundation Security Advisory 2015-100, Mozilla Foundation Security Advisory 2015-101, Mozilla Foundation Security Advisory 2015-105, Mozilla Foundation Security Advisory 2015-106, Mozilla Foundation Security Advisory 2015-110, Mozilla Foundation Security Advisory 2015-111, Mozilla Foundation Security Advisory 2015-112, Mozilla Foundation Security Advisory 2015-113
  • bind9: Debian Security Advisory 3350
  • liblcms1: CVE-2013-4276
  • libldap-2.4-2: Debian Security Advisory 3356
  • libslp1: Debian Security Advisory 3353
  • ssl-cert: Debian bug #773815
