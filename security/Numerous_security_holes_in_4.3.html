Numerous security holes in Tails 4.3 2020-03-10

Tails 4.4 fixes many security issues that affect Tails 4.3. You should upgrade to Tails 4.4 as soon as possible.

  • Tor Browser: Mozilla Foundation Security Advisory 2020-09
  • Thunderbird: Mozilla Foundation Security Advisory 2020-07, Debian Security Advisory 4625
  • Linux: CVE-2019-3016, CVE-2019-14896, CVE-2019-14897, CVE-2020-8428, CVE-2019-19046, CVE-2019-19043
  • cURL: Debian Security Advisory 4633
  • evince: Debian Security Advisory 4624
  • Pillow: Debian Security Advisory 4631
  • ppp: Debian Security Advisory 4632
  • WebKitGTK: Debian Security Advisory 4627
