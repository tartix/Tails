Numerous security holes in Tails 0.23 2014-04-27

Several security holes affect Tails 0.23.

We strongly encourage you to upgrade to Tails 1.0 as soon as possible.

Details

  • Iceweasel and its bundled NSS: Mozilla Foundation Security Advisory 2014-34, Mozilla Foundation Security Advisory 2014-35, Mozilla Foundation Security Advisory 2014-37, Mozilla Foundation Security Advisory 2014-38, Mozilla Foundation Security Advisory 2014-42, Mozilla Foundation Security Advisory 2014-43, Mozilla Foundation Security Advisory 2014-44, Mozilla Foundation Security Advisory 2014-46
  • tor: bug #11464 on Tor Project's Trac
  • openjdk-6: Debian Security Advisory 2912
  • openssl: Debian Security Advisory 2908
  • curl: Debian Security Advisory 2902
  • openssh: Debian Security Advisory 2894
  • libyaml: Debian Security Advisory 2884
  • libyaml-libyaml-perl: Debian Security Advisory 2885
