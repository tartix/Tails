Numerous security holes in Tails 3.13.1 2019-05-04

Several security holes that affect Tails 3.13.1 are now fixed in Tails 3.13.2.

We strongly encourage you to upgrade to Tails 3.13.2 as soon as possible.

  • Tor Browser: Armagadd-on-2.0 (#16694)
  • thunderbird: Debian Security Advisory 4420
  • wget: Debian Security Advisory 4425
  • samba: Debian Security Advisory 4427
  • systemd: Debian Security Advisory 4428
  • wpa: Debian Security Advisory 4430
  • libssh2: Debian Security Advisory 4431
  • ghostscript: Debian Security Advisory 4432
  • libpng1.6: Debian Security Advisory 4435
  • imagemagick: Debian Security Advisory 4436
  • gst-plugins-base1.0: Debian Security Advisory 4437
