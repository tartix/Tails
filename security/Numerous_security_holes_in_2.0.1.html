Numerous security holes in Tails 2.0.1 2016-03-06

Several security holes that affect Tails 2.0.1 are now fixed in Tails 2.2.

We strongly encourage you to upgrade to Tails 2.2 as soon as possible.

Details

  • Tor Browser: Mozilla Foundation Security Advisory 2016-16, Mozilla Foundation Security Advisory 2016-17, Mozilla Foundation Security Advisory 2016-20, Mozilla Foundation Security Advisory 2016-21, Mozilla Foundation Security Advisory 2016-23, Mozilla Foundation Security Advisory 2016-24, Mozilla Foundation Security Advisory 2016-25, Mozilla Foundation Security Advisory 2016-27, Mozilla Foundation Security Advisory 2016-28, Mozilla Foundation Security Advisory 2016-31, Mozilla Foundation Security Advisory 2016-34, Mozilla Foundation Security Advisory 2016-35, Mozilla Foundation Security Advisory 2016-37
  • graphite2: Debian Security Advisory 3479
  • glibc: Debian Security Advisory 3481
  • libreoffice: Debian Security Advisory 3482
  • cpio: Debian Security Advisory 3483
  • libssh2: Debian Security Advisory 3487
  • pillow: Debian Security Advisory 3499
  • openssl: Debian Security Advisory 3500
  • perl: Debian Security Advisory 3501
  • linux: Debian Security Advisory 3503
  • libav: Debian Security Advisory 3506
  • jasper: Debian Security Advisory 3508
