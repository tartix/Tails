Numerous security holes in Tails 0.12.1 2012-09-16

Several security holes affect Tails 0.12.1.

We strongly urge you to upgrade to Tails 0.13 as soon as possible in case you are still using an older version.

Details

  • openjdk (Debian Security Advisory 2507)
  • pidgin (Debian Security Advisory 2509)
  • iceweasel (Debian Security Advisory 2514, MSFA 2012-57, MSFA 2012-58, MSFA 2012-61, MSFA 2012-62, MSFA 2012-63, MSFA 2012-65, MSFA 2012-67, MSFA 2012-69, MSFA 2012-70, MSFA 2012-72
  • isc-dhcp (Debian Security Advisory 2516, Debian Security Advisory 2519)
  • bind9 (Debian Security Advisory 2517, Debian Security Advisory 2547)
  • krb5 (Debian Security Advisory 2518)
  • openoffice.org (Debian Security Advisory 2520)
  • libxml2 (Debian Security Advisory 2521)
  • expat (Debian Security Advisory 2525)
  • libotr (Debian Security Advisory 2526)
  • tor (Debian Security Advisory 2548)
