Numerous security holes in Tails 3.10.1 2018-12-10

Several security holes that affect Tails 3.10.1 are now fixed in Tails 3.11.

We strongly encourage you to upgrade to Tails 3.11 as soon as possible.

  • Tor Browser: Mozilla Foundation Security Advisory 2018-30
  • Thunderbird: Mozilla Foundation Security Advisory 2018-28
  • Linux:
    • CVE-2017-5715
    • CVE-2017-5753
    • CVE-2018-12896
    • CVE-2018-13053
    • CVE-2018-17972
    • CVE-2018-18281
    • CVE-2018-18445
    • CVE-2018-18710
  • cURL:
    • Debian Security Advisory 4331
  • Ghostscript:
    • Debian Security Advisory 4336
    • Debian Security Advisory 4346
  • Perl:
    • Debian Security Advisory 4347
  • Policykit:
    • Debian Security Advisory 4350
  • Samba:
    • Debian Security Advisory 4345
  • OpenSSL:
    • Debian Security Advisory 4348
  • libtiff:
    • Debian Security Advisory 4349
