Numerous security holes in Tails 3.16 2019-09-19

Tails 4.0 fixes many security issues that affect Tails 3.16. You should upgrade to Tails 4.0 as soon as possible.

  • Tor Browser: Mozilla Foundation Security Advisory 2019-33
  • Linux: CVE-2019-15118, CVE-2019-15117, CVE-2019-16714, CVE-2019-14835, CVE-2019-15504, CVE-2019-14821, CVE-2019-14814, CVE-2019-14815, CVE-2019-14816, CVE-2019-15505, and CVE-2019-15538
  • CUPS CVE-2019-8696, CVE-2019-8675, and more security fixes
  • GnuPG: mitigate the certificates flooding attack
  • e2fsprogs Debian Security Advisory 4535
  • ghostscript: Debian Security Advisory 4518, Debian Security Advisory 4499
  • WebKitGTK: Debian Security Advisory 4515
  • Pango: Debian Security Advisory 4496
  • ffmpeg: Debian Security Advisory 4502
  • expat: Debian Security Advisory 4530
  • GLib: CVE-2019-13012
  • libmariadb3: numerous vulnerabilities
  • NSS: CVE-2019-11719, CVE-2019-11727, CVE-2019-11729).
  • LibreOffice: Debian Security Advisory 4519, Debian Security Advisory 4501, Debian Security Advisory 4483, and CVE-2019-9848
  • Samba: Debian Security Advisory 4513
  • OpenSSL: Debian Security Advisory 4539
  • libxslt: CVE-2019-11068, CVE-2019-13117, and CVE-2019-13118
  • zeromq3: Debian Security Advisory 4477
  • patch: Debian Security Advisory 4489
  • Thunderbird: Debian Security Advisory 4523, Debian Security Advisory 4482
  • wpasupplicant: Debian Security Advisory 4538
  • IBus: Debian Security Advisory 4525
  • sudo: Debian Security Advisory 4543
