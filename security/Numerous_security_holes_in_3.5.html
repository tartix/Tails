Numerous security holes in Tails 3.5 2018-03-13

Several security holes that affect Tails 3.5 are now fixed in Tails 3.6.

We strongly encourage you to upgrade to Tails 3.6 as soon as possible.

  • Tor Browser: MSFA 2018-07
  • linux: CVE-2018-5750, CVE-2018-5332, CVE-2018-5333, CVE-2018-5344, CVE-2018-5332, CVE-2018-5333, CVE-2017-5753, CVE-2018-6927, CVE-2018-1000004, CVE-2018-1000028, CVE-2018-5344,
  • poppler: Debian Security Advisory 4097
  • curl: Debian Security Advisory 4098
  • ffmpeg: Debian Security Advisory 4099
  • tiff: Debian Security Advisory 4100
  • thunderbird: Debian Security Advisory 4102
  • p7zip: Debian Security Advisory 4104
  • libtasn1: Debian Security Advisory 4106
  • libreoffice: Debian Security Advisory 4111
  • libvorbis: Debian Security Advisory 4113
  • gcc-6: Debian Security Advisory 4121
  • wavpack: Debian Security Advisory 4125
  • libvpx: Debian Security Advisory 4132
  • isc-dhcp: Debian Security Advisory 4133
  • util-linux: Debian Security Advisory 4134
