Numerous security holes in Tails 0.10 2012-01-27

The following security holes affect Tails 0.10.

We strongly urge you to upgrade to Tails 0.10.1 as soon as possible in case you are still using an older version.

Details

  • Linux kernel (CVE-2012-0056, CVE-2012-0207, CVE-2011-4127, CVE-2011-4622)
  • foomatic-filters (Debian Security Advisory 2380)
  • t1lib (Debian Security Advisory 2388)
  • openssl (Debian Security Advisory 2390, Debian Security Advisory 2392)
  • libxml2 (Debian Security Advisory 2394)
