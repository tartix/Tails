Numerous security holes in Tails 3.0 2017-07-02

Several security holes that affect Tails 3.0 are now fixed in Tails 3.0.1.

We strongly encourage you to upgrade to Tails 3.0.1 as soon as possible.

  • tor: TROVE-2017-006 (aka. bug #22753 on Tor Project's Trac and 2017-0377)
  • Linux: Debian Security Advisory 3886, including 2017-1000364 aka. Stack Clash
  • libc: Debian Security Advisory 3887 (aka. Stack Clash)
  • libexpat1: Debian Security Advisory 3898
  • libgcrypt20: Debian Security Advisory 3901
  • libgnutls30: Debian Security Advisory 3884
