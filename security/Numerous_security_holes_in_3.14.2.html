Numerous security holes in Tails 3.14.2 2019-07-06

Several security holes that affect Tails 3.14.2 are now fixed in Tails 3.15.

We strongly encourage you to upgrade to Tails 3.15 as soon as possible.

  • Tor Browser: Mozilla Foundation Security Advisory 2019-22
  • Thunderbird: Mozilla Foundation Security Advisory 2019-20
  • Expat: Debian Security Advisory 4472
  • OpenSSL 1.0: Debian Security Advisory 4475
  • OpenSSL: Debian Security Advisory 4475
  • Vim: Debian Security Advisory 4467
