Numerous security holes in Tails 0.11 2012-06-11

The following security holes affect Tails 0.11.

We strongly urge you to upgrade to Tails 0.12 as soon as possible in case you are still using an older version.

Details

  • Linux kernel (Debian Security Advisory 2469)
  • pidgin-otr (Debian Security Advisory 2476)
  • libgdata (Debian Security Advisory 2482)
  • openssl (Debian Security Advisory 2475)
  • iceweasel (Debian Security Advisory 2488)
  • nss (Debian Security Advisory 2490)
  • openoffice.org (Debian Security Advisory 2487, Debian Security Advisory 2473)
  • sudo (Debian Security Advisory 2478)
  • libxml2 (Debian Security Advisory 2479)
  • bind9 (Debian Security Advisory 2486)
