Numerous security holes in Tails 4.8 2020-07-26

Tails 4.9 fixes many security issues that affect Tails 4.8. You should upgrade to Tails 4.9 as soon as possible.

  • Tor Browser: Mozilla Foundation Security Advisory 2020-31
  • Thunderbird: Mozilla Foundation Security Advisory 2020-29
  • Linux: CVE-2020-10768, CVE-2020-10767, CVE-2020-10766, CVE-2020-13974, CVE-2020-15852, CVE-2020-10781, CVE-2020-10781, CVE-2020-15780
  • Evolution Data Server: Debian Security Advisory 4725
  • FFmpeg: Debian Security Advisory 4722
  • ImageMagick: Debian Security Advisory 4712
  • NSS: Debian Security Advisory 4726
  • OpenMPT: Debian Security Advisory 4729
  • WebKitGTK: Debian Security Advisory 4724
