Numerous security holes in Tails 0.17.1 2013-04-06

Several security holes affect Tails 0.17.1.

We strongly urge you to upgrade to Tails 0.17.2 as soon as possible in case you are still using an older version.

Details

  • Iceweasel (MFSA 2013-30, MFSA 2013-31, MFSA 2013-32, MFSA 2013-34, MFSA 2013-35, MFSA 2013-36, MFSA 2013-38, MFSA 2013-40)
  • Linux (CVE-2013-0913, CVE-2013-1796, CVE-2013-1797, CVE-2013-1798, CVE-2013-1767, CVE-2013-0231, CVE-2013-0311, CVE-2013-1848, CVE-2013-1792, CVE-2013-1860, CVE-2013-0914, CVE-2013-2546, CVE-2013-2547, CVE-2013-2548, CVE-2012-5517, CVE-2012-6548, CVE-2012-6549)
  • libxml2 (Debian Security Advisory 2652)
  • libdns (Debian Security Advisory 2656)
  • libxslt (Debian Security Advisory 2654)
