Numerous security holes in Tails 3.15 2019-08-31

Several security holes that affect Tails 3.15 are now fixed in Tails 3.16.

We strongly encourage you to upgrade to Tails 3.16 as soon as possible.

  • Tor Browser: Mozilla Foundation Security Advisory 2019-27
  • Thunderbird: Mozilla Foundation Security Advisory 2019-23
  • Linux: CVE-2019-13272, CVE-2019-13233, CVE-2019-12817, CVE-2019-12984, CVE-2019-1999, CVE-2018-20836, CVE-2019-13631, CVE-2019-14283, CVE-2019-10638, CVE-2019-14284, CVE-2019-10207, CVE-2019-13648, CVE-2019-1125
  • LibreOffice: Debian Security Advisory 4483, Debian Security Advisory 4501
  • Ghostscript: Debian Security Advisory 4499
  • Patch: Debian Security Advisory 4489
  • nghttp2: Debian Security Advisory 4511
