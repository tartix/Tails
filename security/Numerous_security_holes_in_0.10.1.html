Numerous security holes in Tails 0.10.1 2012-03-01

The following security holes affect Tails 0.10.1.

We strongly urge you to upgrade to Tails 0.10.2 as soon as possible in case you are still using an older version.

Details

  • icu (Debian Security Advisory 2397)
  • curl (Debian Security Advisory 2398)
  • iceweasel (Debian Security Advisory 2400)
  • libpng (Debian Security Advisory 2410)
  • libvorbis (Debian Security Advisory 2412)
  • libarchive (Debian Security Advisory 2413)
  • libmodplug (Debian Security Advisory 2415)
  • libxml2 (Debian Security Advisory 2417)
  • file (Debian Security Advisory 2422)
