Numerous security holes in Tails 0.10.2 2012-04-22

The following security holes affect Tails 0.10.2.

We strongly urge you to upgrade to Tails 0.11 as soon as possible in case you are still using an older version.

Details

  • iceweasel (Debian Security Advisory 2457, Debian Security Advisory 2433)
  • openssl (Debian Security Advisory 2454)
  • gimp (Debian Security Advisory 2426)
  • imagemagick (Debian Security Advisory 2427)
  • freetype (Debian Security Advisory 2428)
  • libpng (Debian Security Advisory 2439, Debian Security Advisory 2446)
  • libtasn1-3 (Debian Security Advisory 2440)
  • gnutls26 (Debian Security Advisory 2441)
  • tiff (Debian Security Advisory 2447)
