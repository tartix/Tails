Known security vulnerabilities in Tails 4.17 2021-04-17

Tails 4.18 fixes known security vulnerabilities in 4.17. You should upgrade as soon as possible.

If you are interested in learning more about the security vulnerabilities fixed in this release, you can refer to:

  • The release notes of Tor Browser and other applications linked from the release notes of Tails 4.18.

  • The list of recent Debian security advisories published since Tails 4.17.
