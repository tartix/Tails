Improve the Tails user interface

You can improve the Tails user interface and user experience.

  1. Known tasks
  2. Other ideas
  3. Related pages
  4. Resources for user research

Known tasks

The issues labeled T:User interface design on GitLab correspond to things that we would like to see improved in the Tails user interface.

Other ideas

If you are a UI or HCI person, you can surely find other ideas to improve Tails! Even if you don't implement your suggestions yourself, create issues with your ideas so that others can benefit from your insight.
